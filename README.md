# Terminal Portfolio v1.2

> A cinematic, terminal-driven portfolio OS.

![Terminal Portfolio](https://img.shields.io/badge/Status-Live-success) ![License](https://img.shields.io/badge/License-MIT-blue)

A virtual operating system running in the browser, designed to showcase projects and skills through an interactive command-line interface. Built at the intersection of pixels and logic.

## üåü Features

- **Interactive CLI**: Fully functional terminal with command history, tab completion, and file system navigation.
- **Virtual File System (VFS)**: Browse directories, read files, and explore project dossiers.
- **Cinematic Visuals**: Retro CRT effects, scanlines, and immersive sound design.
- **Project Showcase**: Detailed markdown rendering for project portfolios.
- **Hidden Secrets**: Easter eggs and hidden commands for the curious explorer.

## üöÄ Getting Started

### Local Development

1.  Clone the repository:
    ```bash
    git clone https://github.com/Ashborn-047/terminal-portfolio.git
    cd terminal-portfolio
    ```

2.  Start a local server (requires Node.js):
    ```bash
    npx http-server . -o
    ```

3.  Open `http://localhost:8080` in your browser.
